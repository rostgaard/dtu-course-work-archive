The relation between sending end and receiving end quantities are given as
%
\begin{subequations}\label{eq:GenericTwoPort}
\begin{align}
  V_s &= AV_r + BI_r \\
  I_s &= CV_r + DI_r 
\end{align}
\end{subequations}%
\nomenclature{$V$}{Steady-state voltage phasor\nomunit{V}}%
\nomenclature{$I$}{Steady-state current phasor\nomunit{A}}%
\nomenclature{$._r$}{Subscript refers to receiving end quantity}%
\nomenclature{$._s$}{Subscript refers to sending end quantity}%
%
The first time a parameter or variable occur that should be included in the list
of symbols write:

\verb+\nomenclature{$<symbol>$}{Steady-state voltage phasor\nomunit{<unit>}}+
